## About The Project

Machine Learning Model on "Climate Change Forecast of Delhi" based on steps and techniques used by Leandro Rabelo (International Market Specialist at Usina Coruripe Macei√≥, State of Alagoas, Brazil) in his Kaggle project.  The technique known as time series analysis is used to obtain very useful results and statistics using the time series dataset of climate change forecasts.   

Results are visually presented with the help of various powerful SciPy libraries such as Matplotlib and Numpy. One of the biggest help we can get from here is that we can predict climate change like floods using Deep learning models.
